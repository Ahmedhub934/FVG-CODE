# FVG [atrader934]

A TradingView Pine Script indicator that detects and displays Fair Value Gaps (FVGs) on any chart or timeframe.

How to Use
1. Open this repo and copy the code.
2. Paste it into the Pine Editor in TradingView.
3. Add to chart.
